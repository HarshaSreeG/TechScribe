# Tech Scribe — Team Showcase Website

A team showcase website built with HTML, CSS, and vanilla JavaScript. Features an interactive world map, animated travel paths, a hero image slider, team profile cards, and a past work gallery.

## ✨ Features

- **Hero Slider** — 3 slides with auto-advance every 5 seconds
- **About Section** — Team mission with Innovation, Collaboration, Excellence
- **Team Profiles** — Scrollable cards with avatar, skills, and interests
- **Interactive Map** — Leaflet.js map with animated lines from each member's hometown to University of Cincinnati
- **Past Work Gallery** — Grid of completed project cards
- **Responsive** — Works on mobile with hamburger menu
